# HS カメラ 予約システム

## 概要

本システムは、研究室で共有されているハイパースペクトルカメラ（HS カメラ）の予約管理を行うための Web アプリケーションです。複数の班が 1 台のカメラを利用するため、使用状況を管理・保存できる仕組みを提供します。管理者は各班の予約・使用状況を確認することが可能です．

## 使用技術

- Next.js
- Firebase
- Tailwind CSS

## デプロイ先

- Vercel
